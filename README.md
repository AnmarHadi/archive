# نظام أرشفة المراسلات والكتب الرسمية

نظام متكامل لإدارة وأرشفة المراسلات الواردة والصادرة والكتب الرسمية، مبني باستخدام تقنيات الويب الحديثة.

## المميزات

- 📊 **لوحة تحكم رئيسية** - إحصائيات شاملة وعرض آخر المراسلات
- 📥 **إدارة الوارد** - تسجيل وتتبع المراسلات الواردة
- 📤 **إدارة الصادر** - تسجيل وتتبع المراسلات الصادرة
- 🔍 **بحث متقدم** - بحث بمعايير متعددة (الحالة، الأولوية، التصنيف)
- 📈 **التقارير والإحصائيات** - رسوم بيانية وتحليلات شاملة
- 🏢 **إدارة الأقسام** - إدارة أقسام المؤسسة ومعلوماتها
- 👥 **إدارة المستخدمين** - إدارة المستخدمين والصلاحيات
- 📝 **سجل النشاط** - تتبع جميع التغييرات على المراسلات
- 🔒 **مستويات السرية** - عام، داخلي، سري، سري للغاية
- 🏷️ **تصنيفات متعددة** - كتاب رسمي، مذكرة، تعميم، قرار، تقرير، عقد
- 🌐 **واجهة عربية** - دعم كامل للغة العربية (RTL)

## التقنيات المستخدمة

- **الإطار:** Next.js 16 (App Router)
- **اللغة:** TypeScript 5
- **التصميم:** Tailwind CSS 4 + shadcn/ui
- **قاعدة البيانات:** SQLite + Prisma ORM
- **الرسوم المتحركة:** Framer Motion
- **الأيقونات:** Lucide React

## متطلبات التشغيل

- Node.js 18+ أو Bun
- npm أو bun أو yarn

## التثبيت والتشغيل

### 1. استنساخ المستودع

```bash
git clone https://github.com/AnmarHadi/archive.git
cd archive
```

### 2. تثبيت المكتبات

```bash
npm install
# أو
bun install
```

### 3. إعداد متغيرات البيئة

أنشئ ملف `.env` في المجلد الرئيسي:

```env
DATABASE_URL="file:./db/custom.db"
```

### 4. إعداد قاعدة البيانات

```bash
npx prisma db push
# أو
bun run db:push
```

### 5. تشغيل المشروع

```bash
npm run dev
# أو
bun run dev
```

### 6. تهيئة البيانات التجريبية

بعد فتح التطبيق في المتصفح `http://localhost:3000`، اضغط على زر **"تهيئة البيانات التجريبية"** في القائمة الجانبية لملء النظام ببيانات تجريبية.
